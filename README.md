## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.  

# 


## Installation

```bash

$ npm install
```

## Running the app

```bash
# development up BD
$ docker-compose up -d


# watch mode dev
$ docker-compose up -d
$ npm run start:dev

# production mode
$ npm run start:prod
```
